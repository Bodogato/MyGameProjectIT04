Вітаю в Мілліонер. Цей файл являю собою гайд для заповнення питань в міліонері. 
Ви можете бачити папку з назвою Questions. В ній зберігаютться всі види питань. 
Всього типів питань 3 - по рівню складності. 
1 - легкі
2 - трошки складніші 
3 - тяжкі 
Кожне питання представляє собою 4 відповіді, і лише одна з них є правильную. 
Правильну відповідь завжди стоїть перша, а всі інші - не вірні. 
Отже якщо ви хочете задати правильну відповідь замініть перший рядок. 
Приклад як виглядає файл:
Кто такий Энштейн? <- питання
Фізик <- приавильний варіант
Математик <- неправильний варіант
Хімік <- неправильний варіант
Філософ <- неправильний варіант
